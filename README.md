# color
인프런 김영한님의 스프링 MVC, JPA 강의를 듣고 만든 게시판입니다.

### ➰ 개발 환경
- Java 11
- Jdk 11.0.15
- Springboot 2.7.15
- Mysql
- IntelliJ
- 

### 📍 기능
- 사용자
    - 회원가입 /users/signup 
    - 로그인 /users/login
- 게시판
    - 목록조회 /boards?offset=0&limit=5
    - 단일조회 /boards/1
    - 생성 /boards
    - 수정 /boards/edit/1
    - 삭제 /bards/1
- 회사
    - 목록조회 /companies?offset=0&limit=5
    - 생성 /companies
    - 수정 /companies/edit/1
    - 삭제 /companies/1
